# Mike Ogilvy

> **Moments distilled. Light gathered. Knowledge organized.**

This is my personal digital space, built with **Hugo** and hosted on the edge. It serves as a dual-purpose repository for my academic growth and visual storytelling.

-----

## 📂 Content

### 📓 Personal Notes

A collection of my learning journey and daily explorations, including:

  * **Biological Sciences**: In-depth notes on Genetics, Molecular Biology, and Ecology.
  * **Computational Biology**: Practical guides on bioinformatics, coding, and data analysis.
  * **General Learning**: Daily logs, technical troubleshooting, and language studies.

### 📸 Photography

A dedicated gallery for visual storytelling.

  * High-fidelity captures from my travels.
  * Focused on a minimalist, immersive viewing experience.

## 🛠️ Tech Stack

  * **Framework**: [Hugo](https://gohugo.io/) (PaperMod theme)
  * **Deployment**: GitHub Pages
  * **Media Storage**: Cloudflare R2
  * **Workflow**: Windows / WSL
